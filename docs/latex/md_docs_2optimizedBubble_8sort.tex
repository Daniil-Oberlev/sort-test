\hypertarget{md_docs_2optimizedBubble_8sort_autotoc_md42}{}\doxysection{\texorpdfstring{Алгоритм работы}{Алгоритм работы}}\label{md_docs_2optimizedBubble_8sort_autotoc_md42}

\begin{DoxyEnumerate}
\item Проходит по массиву и сравнивает соседние элементы.
\item Если элементы находятся в неправильном порядке, они меняются местами.
\item Если в ходе прохода не было никаких обменов (массив отсортирован), выполнение прерывается.
\item Процесс продолжается до тех пор, пока массив не будет отсортирован.
\end{DoxyEnumerate}\hypertarget{md_docs_2optimizedBubble_8sort_autotoc_md43}{}\doxysection{\texorpdfstring{Примечения}{Примечения}}\label{md_docs_2optimizedBubble_8sort_autotoc_md43}

\begin{DoxyItemize}
\item Алгоритм {\ttfamily optimized\+Bubble\+Sort} более эффективен, чем обычная пузырьковая сортировка, для почти отсортированных массивов.
\item Тем не менее, в худшем случае временная сложность остается O(n²). 
\end{DoxyItemize}
