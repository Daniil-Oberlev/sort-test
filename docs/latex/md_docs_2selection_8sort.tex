\hypertarget{md_docs_2selection_8sort_autotoc_md56}{}\doxysection{\texorpdfstring{Алгоритм работы}{Алгоритм работы}}\label{md_docs_2selection_8sort_autotoc_md56}

\begin{DoxyEnumerate}
\item Проходит по массиву и находит минимальный (или максимальный) элемент в несортированной части массива.
\item Меняет местами найденный элемент с первым элементом несортированной части.
\item Повторяет процесс для оставшейся части массива, пока весь массив не будет отсортирован.
\end{DoxyEnumerate}\hypertarget{md_docs_2selection_8sort_autotoc_md57}{}\doxysection{\texorpdfstring{Примечения}{Примечения}}\label{md_docs_2selection_8sort_autotoc_md57}

\begin{DoxyItemize}
\item Алгоритм сортировки выбором имеет временную сложность O(n²) и не является самым эффективным для больших массивов по сравнению с более современными алгоритмами.
\item Подходит для обучения и работы с небольшими наборами данных. 
\end{DoxyItemize}
