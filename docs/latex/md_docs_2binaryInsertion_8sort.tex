\begin{DoxyEnumerate}
\item Для каждого элемента, начиная со второго, алгоритм находит позицию, куда его можно вставить, используя бинарный поиск.
\item Элементы, которые находятся на позиции вставки и после неё, сдвигаются вправо для освобождения места.
\item Вставляет текущий элемент на найденную позицию.
\item Процесс продолжается для всех элементов массива.

\begin{DoxyItemize}
\item Бинарная сортировка вставками более эффективна, чем обычная сортировка вставками, особенно для больших массивов.
\item Временная сложность составляет O(n log n) в среднем случае, но O(n\texorpdfstring{$^\wedge$}{\string^}2) в худшем случае при использовании вставок.
