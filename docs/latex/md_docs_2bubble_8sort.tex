\hypertarget{md_docs_2bubble_8sort_autotoc_md21}{}\doxysection{\texorpdfstring{Алгоритм работы}{Алгоритм работы}}\label{md_docs_2bubble_8sort_autotoc_md21}

\begin{DoxyEnumerate}
\item Проходит по массиву и сравнивает соседние элементы.
\item Если элементы находятся в неправильном порядке, они меняются местами.
\item Если в ходе прохода не было никаких обменов (массив отсортирован), выполнение прерывается.
\item Процесс продолжается до тех пор, пока массив не будет отсортирован.
\end{DoxyEnumerate}\hypertarget{md_docs_2bubble_8sort_autotoc_md22}{}\doxysection{\texorpdfstring{Примечения}{Примечения}}\label{md_docs_2bubble_8sort_autotoc_md22}

\begin{DoxyItemize}
\item Алгоритм пузырьковой сортировки имеет ограниченную эффективность, особенно для больших массивов, и его средняя и худшая временные сложности составляют O(n²).
\item Он может быть несколько более эффективным для почти отсортированных массивов благодаря возможности преждевременного завершения, если в ходе прохода не было обменов. 
\end{DoxyItemize}
