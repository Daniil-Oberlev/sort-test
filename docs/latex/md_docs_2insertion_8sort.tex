\hypertarget{md_docs_2insertion_8sort_autotoc_md35}{}\doxysection{\texorpdfstring{Алгоритм работы}{Алгоритм работы}}\label{md_docs_2insertion_8sort_autotoc_md35}

\begin{DoxyEnumerate}
\item Начинает с первого элемента и рассматривает его как отсортированную часть.
\item Каждый последующий элемент вставляется в правильную позицию в отсортированной части массива.
\item Процесс продолжается до тех пор, пока все элементы не будут отсортированы.
\end{DoxyEnumerate}\hypertarget{md_docs_2insertion_8sort_autotoc_md36}{}\doxysection{\texorpdfstring{Примечения}{Примечения}}\label{md_docs_2insertion_8sort_autotoc_md36}

\begin{DoxyItemize}
\item Алгоритм сортировки вставками эффективен для небольших массивов и почти отсортированных данных, но неэффективен для больших массивов.
\item Обладает временной сложностью O(n\texorpdfstring{$^\wedge$}{\string^}2) в худшем случае, но O(n) в лучшем. 
\end{DoxyItemize}
